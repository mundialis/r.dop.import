DESCRIPTION

r.dop.import.ni downloads and imports digital orthophotos (DOP) for Niedersachsen (NI) and area of interest.
The data can be used when referencing the source:
id: CC BY 4.0,
name: Namensnennung 4.0 International,
url: https://creativecommons.org/licenses/by/4.0/,
source: (c) LGL-Niedersachsen (LGLN)

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.ni aoi=aoi_NI output=dop_NI -r
