DESCRIPTION

r.dop.import.bw downloads and imports digital orthophotos (DOP) for Baden-Württemberg (BW) and area of interest using the respective WMS and WMS.
The data can be used when referencing the source:
id: dl-by-de/2.0,
name: Datenlizenz Deutschland Namensnennung 2.0,
url: https://www.govdata.de/dl-de/by-2-0,
source: (c) LGL-BW (LGL-BW)

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.bw aoi=aoi_BW output=dop_BW -r
