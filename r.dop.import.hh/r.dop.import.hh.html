DESCRIPTION

r.dop.import.hh downloads and imports digital orthophotos (DOP) for Hamburg (HH) and area of interest. The data "Digitale Orthophotos" is downloaded from Transparenzportal Hamburg and can be used under the specification of the license and referencing the source:
source: GeoSN
license: Datenlizenz Deutschland Namensnennung 2.0,
DOP

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.hh aoi=aoi_HH output=dop_HH -r
