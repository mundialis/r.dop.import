DESCRIPTION

r.dop.import.sn downloads and imports digital orthophotos (DOP) for Sachsen (SN) and area of interest.
The data can be used when referencing the source:
id: dl-by-de/2.0,
name: Datenlizenz Deutschland - Zero - Version 2.0,
url: https://www.govdata.de/dl-de/zero-2-0,
source: (c) Landesamt für Geobasisinformation Sachsen (GeoSN)

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.nw aoi=aoi_SN output=dop_SN -r
