DESCRIPTION

r.dop.import.by downloads and imports digital orthophotos (DOP) for Bayern (BY) and area of interest using the respective WMS.
The data can be used when referencing the source:
id: CC BY 4.0,
name: Namensnennung 4.0 International,
url: https://creativecommons.org/licenses/by/4.0/,
source: (c) Bayerische Vermessungsverwaltung (Bayerische Vermessungsverwaltung)

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.by aoi=aoi_BY output=dop_BY -r
