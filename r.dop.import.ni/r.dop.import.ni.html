DESCRIPTION

r.dop.import.ni downloads and imports digital orthophotos (DOP) for Niedersachsen (NI) and area of interest. The data "Digitale Orthophotos" is downloaded from OpenGeoData Niedersachsen and can be used under the specification of the license: © GeoBasis-DE/LGLN 2025, Creative Commons (CC-BY 4.0), DOP.

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.ni aoi=aoi_NI output=dop_NI -r
